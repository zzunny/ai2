APP Inventor 2

2017 GSITM 모바일 워킹그룹

'APP Inventor 2' 란 무엇인가?

MIT에서 교육용으로 고안하여 개발한 Open Block Library의 블럭 프로그래밍 기술을 이용하여 웹브라우져에서 안드로이드용 스마트폰 응용프로그램(앱)을 구축할 수 있는 클라우드기반의 앱 개발도구로 코딩없이 개발이 가능하다.

AI 2 메뉴설명

User Interface - 사용자가 제어/설정 하는 개체

Button - 일반적인, 터치하면 명령을 수행하는 개체

TextBox - 문자열을 써 넣을 수 있는 공간입니다.

ListView - 목록을 표시하게 하는 개체

DatePicker - 날짜를 사용자가 설정하게 해 주는 개체

TimePicker - 시간을 사용자가 설정하게 해 주는 개체

CheckBox - 체크하여 항목을 선택하는데 사용합니다.

Label - 화면에 일반적인 문자열을 표시하는데 사용합니다.

ListPicker - 목록에서 사용자가 아이템을 선택하게 해 주는 개체

Slider - 누른 채로 좌우로 움직여서 값을 설정하게 해 주는 개체

PasswordTextBox - 비밀번호 입력할 수 있는 공간입니다.

Notifier - 경고창, 메세지등을 표시하게 해 주는 개체

Image - 화면에 이미지를 표시해 주는 개체

WebViewer - 화면에 웹사이트를 표시해 주는 개체

Spinner - 목록 중에 사용자가 아이템을 선택하게 해 주는 개체

Layout - 화면의 개체를 정렬할 때 사용

TableArrangement - 표의 형식으로 행과 열에 따라 개체를 정렬

VerticalArrangement - 무조건 세로로 개체들을 정렬

HorizontalArrangement - 무조건 가로로 개체들을 정렬

Media - 동영상, 소리, 이미지에 관한 개체

Player - 소리를 재생하는 개체

Sound - 소리 파일을 재생하는 개체

TextToSpeech - 문자열을 소리로 변환하여 소리를 재생하는 개체

YandexTranslate - 여러가지 언어로 번역해 주는 개체

VideoPlayer - 영상을 재생하는 개체

Camcorder - 영상을 녹화하는 개체

Camera - 사진을 촬영하는 개체

SpeechRecognizer - 음성을 텍스트로 바꾸어주는 개체

ImagePicker - 갤러리에서 이미지를 선택할 수 있는 개체

SoundRecorder - 소리를 녹음하는 개체

Drawing and Animation - 그리기와 애니메이션에 관한 개체

Canvas - 자유롭게 그리거나 애니메이션이 이루어지는 공간

ImageSprite - Canvas 안에서 움직이는 이미지 개체

Ball - Canvas 안에서 움직이는 공 개체

Sensors - 휴대폰의 센서들에 관한 개체

Clock - 시간에 관한 명령을 처리하는 개체

GyroscopeSensor - 자이로스코프 센서에 관한 명령을 처리

[자이로스코프에 대한 정보 : https://ko.wikipedia.org/wiki/자이로스코프]

BarcodeScanner - 바코드를 읽는 것에 관한 명령을 처리

LocationSensor - GPS를 통한 현재 위치 파악에 관한 명령을 처리

OrientationSensor - 방위나 각도에 관한 명령을 처리

NearField - NFC에 관한 명령을 처리

[NFC에 대한 정보 : https://ko.wikipedia.org/wiki/근거리_무선_통신]

AccelerometerSensor - 휴대폰의 각도에 관한 명령을 처리

ProximitySensor - 휴대폰과 물체와의 거리에 관한 명령을 처리

Social - SNS, 문자, 통화, 이메일 등에 관한 명령을 처리

EmailPicker - 이메일을 선택하게 해 주는 개체

Texting - 문자에 관한 명령을 처리합니다.

PhoneNumberPicker - 휴대폰 번호를 선택하게 해 주는 개체

PhoneCall - 전화를 걸게 해 주는 개체

Sharing - 공유에 관련된 명령을 처리합니다.

Twitter - 트위터에 관련된 명령을 처리합니다.

ContactPicker - 주소록의 항목을 선택하게 해 주는 개체

Storage - 저장이나 파일에 관련된 명령을 처리

FusiontablesControl - 표 형태의 자료에 관련된 명령을 처리

File - 텍스트 파일에 관련된 명령을 처리

TinyWebDB - 정보를 저장할 수 있는 웹 데이터베이스

TinyDB - 정보를 저장할 수 있는 휴대폰 내의 데이터베이스

Connectivity - 정보 교환에 관련된 명령을 처리

BluetoothClient - 블루투스 클라이언트에 관련된 명령을 처리

BluetoothServer - 블루투스 서버에 관련된 명령을 처리

Web - 웹페이지에 관련된 명령을 처리

ActivityStarter - 휴대폰에게 특정한 명령을 수행시킬 때 사용

LEGO®MINDSTORMS® - 레고 Nxt를 제어하는 개체

NxtDrive

NxtColorSensor

NxtLightSensor

NxtSoundSensor

NxtTouchSensor

NxtUltrasonicSensor

NxtDirectCommands

Experimental - 기타 요소

FirebaseDB - 일종의 SNS 서비스인 Firebase에 관한 명령을 수행.
